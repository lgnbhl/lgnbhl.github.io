# Package index

## Swiss Stats Explorer API

The Swiss Stats Explorer is the new official API of the Swiss Federal
Statistical Office.

- [`bfs_get_sse_data()`](https://felixluginbuhl.com/BFS/reference/bfs_get_sse_data.md)
  : Get Data from Swiss Stats Explorer (SSE) API
- [`bfs_get_sse_metadata()`](https://felixluginbuhl.com/BFS/reference/bfs_get_sse_metadata.md)
  : Get Metadata (Codelists) from Swiss Stats Explorer (SSE) API
- [`bfs_get_sse_url()`](https://felixluginbuhl.com/BFS/reference/bfs_get_sse_url.md)
  : Get the URL for the Swiss Stats Explorer (SSE) API based on the BFS
  number
- [`get_xml_name()`](https://felixluginbuhl.com/BFS/reference/get_xml_name.md)
  : Get the name from an XML node for a specified language, with
  fallback

## PXWeb API

PXWEB is an API structure developed to disseminate public statistics.

- [`bfs_get_catalog()`](https://felixluginbuhl.com/BFS/reference/bfs_get_catalog.md)
  : Get the BFS data or table catalog
- [`bfs_get_catalog_data()`](https://felixluginbuhl.com/BFS/reference/bfs_get_catalog_data.md)
  : Get the BFS data catalog
- [`bfs_get_catalog_tables()`](https://felixluginbuhl.com/BFS/reference/bfs_get_catalog_tables.md)
  : Get the BFS tables catalog
- [`bfs_get_data()`](https://felixluginbuhl.com/BFS/reference/bfs_get_data.md)
  : Get BFS data in a given language
- [`bfs_get_metadata()`](https://felixluginbuhl.com/BFS/reference/bfs_get_metadata.md)
  : Get metadata of a BFS data in a given language
- [`bfs_get_data_comments()`](https://felixluginbuhl.com/BFS/reference/bfs_get_data_comments.md)
  : Get the comments/footnotes of a BFS dataset in a given language

## DAM API

- [`bfs_download_asset()`](https://felixluginbuhl.com/BFS/reference/bfs_download_asset.md)

## STAC API

STAC is a specification of files and web services used to describe
geospatial information assets.

- [`bfs_get_catalog_geodata()`](https://felixluginbuhl.com/BFS/reference/bfs_get_catalog_geodata.md)
  : Get the geodata catalog of the Swiss Confederation
- [`bfs_download_geodata()`](https://felixluginbuhl.com/BFS/reference/bfs_download_geodata.md)
  : Download a geographic file from the Swiss geo-portal

## Commune Register

The Swiss official commune register is structured according to cantons
and districts or comparable administrative entities. Various federal,
cantonal and communal governments and private businesses use this
register when identifying and referring to communes.

- [`register_dic`](https://felixluginbuhl.com/BFS/reference/register_dic.md)
  : Swiss official commune register Dictionary
- [`register_bzn`](https://felixluginbuhl.com/BFS/reference/register_bzn.md)
  : Swiss official commune register BZN
- [`register_gde`](https://felixluginbuhl.com/BFS/reference/register_gde.md)
  : Swiss official commune register GDE
- [`register_gde_other`](https://felixluginbuhl.com/BFS/reference/register_gde_other.md)
  : Swiss official commune register GDE Other
- [`register_kt`](https://felixluginbuhl.com/BFS/reference/register_kt.md)
  : Swiss official commune register KT
- [`register_kt_seeanteile`](https://felixluginbuhl.com/BFS/reference/register_kt_seeanteile.md)
  : Swiss official commune register KT Seeanteile
