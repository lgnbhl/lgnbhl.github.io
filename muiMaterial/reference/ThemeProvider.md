# ThemeProvider

Visit the documentation at https://mui.com/api/Theme-Provider/ for more
information.

## Usage

``` r
ThemeProvider(...)
```

## Arguments

- ...:

  args to pass to element

## Value

Object with \`shiny.tag\` class suitable for use in the UI of a Shiny
app.

## Details

Custom ThemeProvider built with MuiThemeProvider. See
'js/src/ThemeProvider.tsx'.
