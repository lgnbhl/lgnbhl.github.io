# SpeedDialIcon

<https://mui.com/material-ui/api/speed-dial-icon/>

## Usage

``` r
SpeedDialIcon(...)
```

## Arguments

- ...:

  Props to pass to the component.

## Value

Object with `shiny.tag` class suitable for use in the UI of a Shiny app.

## Details

- classes `object`  
  Default is NA Override or extend the styles applied to the
  component.See CSS classes API below for more details.

- icon `node`  
  Default is NA The icon to display.

- openIcon `node`  
  Default is NA The icon to display in the SpeedDial Floating Action
  Button when the SpeedDial is open.

- sx `Array<func| object| bool>| func| object`  
  Default is NA The system prop that allows defining system overrides as
  well as additional CSS styles.See the `sx` page for more details.
