# Export the data structure of a dataset

Calls the I14Y public API endpoint
`/datasets/{datasetId}/structures/exports/{dataFormat}`. The endpoint
returns a SHACL/RDF graph in one of the supported linked-data formats.
Note that many datasets do not have a published structure and will
return an empty result.

## Usage

``` r
i14y_get_dataset_structure(id = NULL, format = "JsonLd")
```

## Arguments

- id:

  string. The UUID of the dataset.

- format:

  string. The export format. One of "JsonLd" (default), "Ttl" or "Rdf".

## Value

For \`format = "JsonLd"\`, the parsed JSON-LD graph as a list. For other
formats, the raw response body as a character string. \`NULL\` when
offline.
