# GaugeValueText

<https://mui.com/x/api/charts/gauge-value-text/>

## Usage

``` r
GaugeValueText(...)
```

## Arguments

- ...:

  Props to pass to the component.

## Value

Object with `shiny.tag` class suitable for use in the UI of a Shiny app.

## Details

- text `func, string`\
  Default is NA The text to display in the gauge. Can be a string or a
  function receiving the value and returning a string.
