# ChartsToolbar

<https://mui.com/x/api/charts/toolbar/>

Wraps the `Toolbar` component of `@mui/x-charts`. It is exported as
`ChartsToolbar` in R because `muiMaterial` already exports Material UI's
`Toolbar` (the AppBar one) and the two packages are commonly attached
together.

## Usage

``` r
ChartsToolbar(...)
```

## Arguments

- ...:

  Props to pass to the component.

## Value

Object with `shiny.tag` class suitable for use in the UI of a Shiny app.
